## Description

[Pexies](https://github.com/canersevince/pexies) Pexies is a personal photo gallery built on pexels api. You can see curated pictures, randomized pictures and add them to your favourites.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
